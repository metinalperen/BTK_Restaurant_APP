## 🖥️ Teknolojiler

- **Framework**: Spring Boot 3.3.2
- **Language**: Java 21
- **Database**: PostgreSQL 42.7.7
- **Security**: Spring Security + JWT
- **ORM**: Spring Data JPA + Hibernate
- **Validation**: Bean Validation
- **Email**: Spring Mail
- **Scheduling**: Spring Scheduling
- **Build Tool**: Maven
- **Migration**: Flyway

---

## 🚀 Kurulum

```bash
cd backend
mvn spring-boot:run
```
